# OOTD（PWA）— iPhone 使用说明

这个小工具是一个 PWA（网页小程序）。你可以用 iPhone 的 Safari 打开后“添加到主屏幕”，之后就像 App 一样从桌面图标进入使用。数据（衣服图片、标签、搭配）默认保存在手机本地。

## 1. 安装到主屏幕（像 App 一样打开）

1. 用 iPhone 打开 **Safari**
2. 在 Safari 地址栏输入你的链接并打开
3. 点击下方 **分享按钮**（一个方框向上箭头）
4. 选择 **添加到主屏幕**
5. 名称可以改成“穿搭”或“衣柜”，点 **添加**
6. 回到桌面，点击新图标打开即可

> 注意：必须用 Safari 才能添加到主屏幕。微信/QQ 内置浏览器一般不行。

---

## 2. 日常使用流程

### 2.1 添加衣服
1. 点“添加衣服”
2. 选择图片（相册或拍照）
3. 选择分类（帽子 / 围巾 / 上衣 / 裤子 / 鞋）
4. 输入颜色
5. 选择“标签”（下拉可选，也可以选“自定义…”输入）
6. 选择季节（可多选）
7. 点击“保存”

### 2.2 衣柜筛选
在“衣柜”区域可以按：
- 分类
- 颜色（下拉）
- 标签（下拉）
- 季节（多选）

> 颜色/标签如果是你在上传时自定义的，保存后会自动出现在过滤下拉列表里。

### 2.3 当前搭配与保存
- 在衣柜卡片点“加入搭配”，会自动加入对应槽位（上衣/裤子/鞋等）
- “当前搭配”里可以对每个槽位点“选择/清除”
- 点“一键清空”可清空当前搭配
- 在“搭配名称”里输入名字（可不填），点击“保存搭配”
- “保存的搭配”里可以加载/删除

---

## 3. 本地存储说明

- 你的衣服图片、颜色、标签、季节、搭配等数据 **都保存在 iPhone 本机**（浏览器的本地数据库 IndexedDB）
- 这些数据 **不会上传到 GitHub**（GitHub Pages 只是托管网页代码）
- 换手机/清理 Safari 数据/系统空间紧张时，数据可能会丢失
---

## 4. 项目说明

该项目是一个个人简易小工具，记录穿搭组合，方便出门前挑选，解决出门前“今天穿什么的烦恼”。
